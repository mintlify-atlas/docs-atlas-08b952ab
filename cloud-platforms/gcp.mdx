## Overview

Google Cloud Platform (GCP) is Google's comprehensive cloud computing platform offering infrastructure, platform, and software services. For cybersecurity professionals, GCP provides robust security tools including Cloud Security Command Center, Identity and Access Management (IAM), and Cloud Armor for DDoS protection.

<Note>
Days 71-77 of the 90 Days of Cybersecurity program focus on building cloud platform skills. GCP is one of three recommended platforms - choose the one that best fits your career goals.
</Note>

## Key Security Services to Explore

As you work through GCP, focus on these security-relevant services:

- **Cloud Identity and Access Management (IAM)** - Control who has access to which resources
- **Cloud Security Command Center** - Centralized security and risk management
- **Cloud Armor** - DDoS protection and web application firewall
- **VPC Service Controls** - Create security perimeters around GCP resources
- **Cloud Key Management Service (KMS)** - Manage encryption keys
- **Security Health Analytics** - Automatic vulnerability and threat detection
- **Cloud Logging and Monitoring** - Track activity and detect anomalies

## Next Steps

After completing the GCP fundamentals:

1. Practice setting up secure cloud architectures
2. Experiment with security monitoring and logging
3. Complete security-focused quests on Cloud Skills Boost
4. Consider pursuing the [Google Cloud Security Engineer certification](https://cloud.google.com/certification/cloud-security-engineer)

Remember: The goal is to understand cloud security concepts that will transfer to any environment, not to memorize every GCP service.
